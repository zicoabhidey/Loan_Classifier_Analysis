# Loan_Classifier_Analysis

The goal of this project is to check the performance of different classification algorithms on loan data and get a insight on the best classifier. 

### Classifiers:
1. KNN
2. Decision tree
3. SVM (SVC)
4. Logistic Regression
5. Random Forest

### Evaluation Metrics:
1. Jaccard
2. F1-Score
3. Log Loss
